## 🚀 Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Create environment file
cp .env.example .env

# 3. Create admin user
node tools/create-admin.js

# 4. Start server
npm start

# 5. Access website
# Frontend: http://localhost:3000
# Admin: http://localhost:3000/admin
# Login: admin / admin123
```
