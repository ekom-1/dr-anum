## TECHNICAL SPECIFICATIONS

### Image Optimization
- **File Size Targets:**
  - Hero images: < 300KB
  - Section images: < 150KB
  - Thumbnails: < 50KB
  - Icons/Logo: < 10KB

### Responsive Sizes Needed
For each main image, create 3 versions:
- **Desktop:** Full resolution (1920px width)
- **Tablet:** Medium resolution (1024px width)
- **Mobile:** Small resolution (640px width)

### Color Accuracy
All images must maintain color consistency:
- Rose Gold: #D4A574
- Cream/Ivory: #FAF7F5
- Nude/Beige: #E8DDD3
- White: #FFFFFF
- Charcoal: #2C2C2C

### Alt Text Format
Every image needs descriptive alt text:
- Format: "[Description] - Dr. Anum Dermatology"
- Example: "Professional dermatologist examining patient's skin - Dr. Anum Dermatology"
